# AmneziaWG-EzServer-OpenWRT

## Что это за скрипт?

Это полностью автоматизированный установщик и конфигуратор AmneziaWG для OpenWRT, который превращает роутер в готовый VPN-сервер за один запуск, без ручного ковыряния в UCI и боли с firewall.

Скрипт проверяет систему и зависимости, при необходимости устанавливает AmneziaWG, генерирует уникальные анти-DPI параметры и криптографические ключи, создаёт и настраивает VPN-интерфейс с пиром через UCI, автоматически конфигурирует firewall и маршрутизацию, применяет изменения с перезапуском сети, опционально интегрирует podkop для контроля nftables и в конце выдаёт полностью готовый клиентский конфиг для подключения.

## Запуск:
```
sh -c "$(wget -O - https://raw.githubusercontent.com/tankionline2005/AmneziaWG-EzServer-OpenWRT/main/awg-ezserver-openwrt.sh)"
```
