# AmneziaWG-EzServer-OpenWRT

## Что это за скрипт?

Это полностью автоматизированный установщик и конфигуратор AmneziaWG для OpenWRT, который превращает роутер в готовый VPN-сервер за один запуск, без ручного ковыряния в UCI и боли с firewall.

Скрипт проверяет систему и зависимости, при необходимости устанавливает AmneziaWG, генерирует уникальные анти-DPI параметры и криптографические ключи, создаёт и настраивает VPN-интерфейс с пиром через UCI, автоматически конфигурирует firewall и маршрутизацию, применяет изменения с перезапуском сети, опционально интегрирует podkop для контроля nftables и в конце выдаёт полностью готовый клиентский конфиг для подключения.

## Зачем это нужно?

VPN-сервер на роутере — это ваш личный защищённый “вход домой” из любой точки: подключаетесь с телефона на улице и оказываетесь внутри своей локальной сети, как будто вы физически находитесь дома. Вы получаете доступ ко всем внутренним сервисам (например, TgWs или Zapret), можете гнать трафик через реальный VPN за границей через podkop — обходить блокировки, не паля реальный IP российским приложениям. Это, вероятно, один из лучших сценариев использования VPN на телефоне =)

---
> [!WARNING]  
> Во время работы, скрипт у вас спросит "Установить скрипты перенаправления трафика из awg1 в podkop?". Соглашайтесь, если вы хотите что бы подключаясь к роутеру вы могли пользоваться ВПН, который установлен в podkop. Если вы откажетесь, трафик будет идти напрямую в интернет через wan, и никакие блокировки вы не обойдёте! (Для этого будет создан mark-watcher.sh)

## Запуск:
```
sh -c "$(wget -O - https://raw.githubusercontent.com/tankionline2005/AmneziaWG-EzServer-OpenWRT/main/awg-ezserver-openwrt.sh)"
```

> [!WARNING]  
> После того как скрипт закончит работу, обязательно перезагрузите роутер, иначе mark-watcher.sh не запустится!
```
reboot
```
